# WeatherIsNice

## 프로젝트 소개
iPhone의 기본 "날씨"앱 클론 코딩을 목표로 진행하였습니다.

## 개발 기간
- 24.04.30 ~ 24.05.03

## 사용한 API
- https://openweathermap.org/current
현재의 날씨 값을 받아오는 Weather API
  
- https://openweathermap.org/forecast5
5일간 3시간 간격의 데이터를 받아오는 Forecast API
